# TidyTuesday

[TidyTuesday](https://github.com/rfordatascience/tidytuesday/blob/master/README.md)
is a weekly social data project organised by the [Data Science Learning
Community](https://dslc.io/). This repository contains the R code used to
produce my visualisations of TidyTuesday data. You can follow my attempts on
[LinkedIn](https://www.linkedin.com/in/cormac-monaghan/), or
[BlueSky](https://bsky.app/profile/c-monaghan.bsky.social),
